Local Storage and Session Storage:
由於Storage只能儲存String.我們若想要把object,array等等資料類型存放在Storage內部的話,
需要用到JSON Object. JSON 代表 JavaScript Object Notation, 是一種資料交換格式.

JSON Object 有兩個method :

JSON.stringify(value) :
將value轉換為 JSON String.

JSON.parse(text) :
解析 JSON String, 製作出JSON String描述的JavaScript值或Object.

[1, 2, 3]　→ setItem() → local storage "[1, 2, 3]" → "[1, 2, 3]"
因為變成String所以無法使用forEach, push, pop等等meythod

[1, 2, 3] → JSON.stringify() → JSON String → setItem() → locak storage "JSON String"
→ getItem() → JSON String → JSON.parse () 解析 → [1, 2, 3]
又變回[] array 可以用forEach等等...
